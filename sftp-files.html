SFTP Files

The fast, focused SFTP client for iPhone, iPad, and Mac. Any server you can ssh to, one tap away.

Coming soon to the App Store

Built for SFTP, not buried in menus. If it runs SSH — a Raspberry Pi, a NAS, a VPS, a web host — SFTP Files connects with the same credentials you'd use in a terminal, and gets straight to your files.

What it does

Connects like ssh

Password or private key (Ed25519 & RSA, passphrases supported) — the same sign-in you already use.

Browse fast

Folders first, instant navigation, rename, delete, new folders, and uploads/downloads with live progress.

Mounts in Finder

On Mac, flip one switch and your server appears in Finder's sidebar — every app can open and save files on it.

Built-in console

Run quick commands on the server — ls, df -h, systemctl status — without leaving the app.

Protocol view

See the actual SFTP conversation (SSH_FXP_*) live — great for debugging servers and firewalls.

Private by design

Everything is encrypted over SSH, directly between your device and your server. No cloud, no account.

Universal purchase. Buy SFTP Files Pro (remove ads) once and it applies on iPhone, iPad, and Mac.

Works with OpenSSH on Linux and Raspberry Pi, macOS Remote Login, Synology / QNAP / TrueNAS, and SFTP-only web hosting. Requires iOS 17 or macOS 14.
