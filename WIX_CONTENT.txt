========================================
HOW TO ADD IN WIX
========================================

1. **Create pages:** Home, About
2. **Add sections:** Drag "Strip" elements for each section
3. **Add text:** Use "Text" elements, copy/paste content above
4. **Add buttons:** Use "Button" elements, link to App Store URLs
5. **Add icons:** Use "Icon" elements or emoji in text
6. **Style it:** Click elements to change colors, fonts, spacing

That's it! Should take 10-15 minutes to set up.
